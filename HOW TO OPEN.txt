1. Install the **"200 OK" Web Server** extension from the Chrome Web Store.
2. Launch the extension and **select the folder** containing this website's code.
3. Start the server.
4. Open your browser and navigate to the local address (usually: http://127.0.0.1:8887).
5. Click `index.html` to open the site.

🔹 OPTION 2: Use Live Server in VS Code
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Open **Visual Studio Code**.
2. Use **File > Open Folder** and select the folder with this website's code.
3. Install the **Live Server** extension if not already installed.
4. Right-click on `index.html` → select **"Open with Live Server"**.
5. The site will open automatically in your browser.
